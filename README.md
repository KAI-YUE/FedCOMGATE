# FedCOMGATE

```
Haddadpour, Farzin, Mohammad Mahdi Kamani, Aryan Mokhtari, and Mehrdad Mahdavi. "Federated learning with compression: Unified analysis and sharp guarantees." arXiv preprint arXiv:2007.01154 (2020).
```
